Lockpass is a Password Generator and Checker
-

You have 2 commands : \
--password -g   //Generate a Password \
--password -c   //Check if a password is secure

Setup
-
```bash
git clone https://github.com/M4lo0oo/Lockpass
cd lockpass
```
After Start Project :
```bash
python main.py
```
